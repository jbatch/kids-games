# 🌟 Kids Games

A collection of simple browser games designed for children around age 4. Each game teaches a foundational computer skill — mouse control, keyboard recognition, or spatial reasoning — in a low-pressure, colourful environment with audio and visual feedback.

## Who these games are for

Primarily aimed at **3–5 year olds** who are just starting to use a computer. The games are:

- **Mouse and keyboard only** — no reading required to play
- **Forgiving** — mistakes don't end the game, just encourage trying again
- **Short sessions** — each game can be played in 2–5 minutes
- **Loud and colourful** — big targets, bright colours, satisfying sounds and animations

---

## Games

### 🐭 Mouse Adventure
**Skill: mouse control**
Teaches moving the mouse cursor, single-clicking, double-clicking, and click-and-drag across 5 progressive levels. Great as a first-ever computer activity.

### 🔍 Letter Hunt
**Skill: letter recognition + mouse clicking**
A letter is shown and read aloud. The child finds and clicks the matching letter from a set of colourful tiles. Builds alphabet recognition and reinforces mouse clicking.

### ⌨️ Letter Keyboard
**Skill: keyboard familiarity + letter recognition**
A letter is shown and spoken. The child presses the matching key on a physical keyboard. Introduces the keyboard layout and reinforces letter–key association.

### 📖 Emoji Stories
**Skill: mouse clicking + vocabulary**
Fill-in-the-blank stories with emoji word choices. Each completed story is read aloud. Builds vocabulary and listening comprehension while practising mouse clicks.

### 🪙 Coin Climber
**Skill: arrow keys + spatial awareness**
A 2D platformer. Collect all the coins on each level then reach the door to progress. Levels start very easy (just walk right) and gradually introduce jumping, gaps, and height. Teaches arrow key control and basic platformer intuition across 6 levels.

**Controls:** `←` `→` arrow keys to move, `↑` or `Space` to jump. Also works with WASD.

## Running locally

Requires [Node.js](https://nodejs.org) and [pnpm](https://pnpm.io).

```bash
pnpm install
pnpm dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Building for production

```bash
pnpm build
```

Output goes to `dist/`. Deploy the contents of that folder to any static host (GitHub Pages, Netlify, etc.).
